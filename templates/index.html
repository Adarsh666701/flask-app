Welcome to the Flask Minimal Web App

This web application demonstrates a basic CI/CD pipeline using modern DevOps tools and cloud services.

Main Features

  • Integration with CircleCI for automated testing and deployment
  • Version control using GitHub
  • Deployment on AWS EC2 instances
  • Containerization with Docker
  • Orchestration using Kubernetes

Purpose

The main goal of this project is to provide a minimal, yet functional example of a full DevOps pipeline, so that developers can understand how code flows from source control to production deployment in the cloud.

Getting Started

To run this application locally:

  1. Clone the repository from GitHub.
  2. Install dependencies using pip install -r requirements.txt.
  3. Run the Flask app using flask run.
  4. Open http://localhost:5000 in your browser.

Explore the pipeline setup and deployment scripts to understand the integration of each DevOps tool.
